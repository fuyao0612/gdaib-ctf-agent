# 变更记录

## v0.5.0

- 新增独立普通聊天链：自然语言多轮对话、标准 SSE 流式输出、完整响应降级、停止和重试。
- 新对话默认使用“对话”；只有明确选择“Agent 任务”才创建 Run 并进入计划与验证流程。
- Web 改为浅色聊天优先布局，侧栏可折叠，运行审计改为按需抽屉，输入区保持可见。
- Agent 任务时间线支持 Task Brief、计划、暂停、继续、按序追加指引及重规划状态展示。
- 设置中心新增默认聊天模型、默认回复方式、系统提示词、流式、上下文和界面偏好。
- 完成长对话与 1024×768 至 2048×1152 六种窗口的滚动、遮挡和横向溢出验收。
- 保持历史 Thread、Run、Provider、AgentProfile 和 SQLite 数据兼容。

## v0.4.2

- 修复启动、诊断、Provider 配置和生产检查中的兼容性问题。
